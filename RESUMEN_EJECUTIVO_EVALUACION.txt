PROBLEMAS ENCONTRADOS (7 PRINCIPALES)
═══════════════════════════════════════════════════════════════════════════════

[CRITICO] 1. PUERTO 8000 BLOQUEADO
   - Impacto: Backend no inicia
   - Causa: Procesos escuchando en puerto
   - Solucion: taskkill o usar puerto 8001
   - Tiempo: 30 min

[ALTA] 2. LOGGING NO ESTRUCTURADO
   - Impacto: Dificulta debugging en producción
   - Causa: Sin JSON logging
   - Solucion: Implementar python-json-logger
   - Tiempo: 3-4 horas

[ALTA] 3. FALTA PAGINACION
   - Impacto: Rendimiento degrada con datos
   - Causa: Endpoints retornan todo el dataset
   - Solucion: Implement offset/limit
   - Tiempo: 4-5 horas

[ALTA] 4. SIN CACHE
   - Impacto: Consultas repetidas a BD
   - Causa: No hay Redis
   - Solucion: Implementar caché Redis
   - Tiempo: 6-8 horas

[MEDIA] 5. NLP CON REGEX (FRAGIL)
   - Impacto: Fallos en extracción de entidades
   - Causa: Regex no maneja variaciones
   - Solucion: spaCy o transformers
   - Tiempo: 6-8 horas

[MEDIA] 6. BAJO COVERAGE DE TESTS
   - Impacto: Riesgo de regresiones
   - Causa: ~20% cobertura
   - Solucion: Escribir tests con pytest
   - Tiempo: 10-12 horas

[MEDIA] 7. SIN RATE LIMITING
   - Impacto: Vulnerable a abuse/DOS
   - Causa: No hay limitador
   - Solucion: Implementar slowapi
   - Tiempo: 3 horas
